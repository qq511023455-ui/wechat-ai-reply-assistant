# 微信聊天 AI 回复助手

这是一个基于 Scripting 运行的 `.tsx` 脚本，旨在帮助用户快速分析微信聊天记录，并结合 AI 生成得体、个性化的回复建议。

## ✨ 功能特点

- **便捷导入**：支持在微信聊天窗口中**多选聊天记录**并复制到剪切板。进入脚本 UI 界面后，点击“自动获取剪切板”即可一键导入。
- **智能解析**：脚本会自动分析剪切板内容，提取出**聊天成员、日期和具体的聊天内容**。
- **角色自定义**：
  - 支持从解析出的聊天角色中，自由勾选哪个是**用户自己**。
  - 可手动填写/备注**对方的身份**（如：老板、客户、相亲对象、朋友等）。
- **心声补充**：支持输入用户对该聊天背景的**真实想法或隐藏意图**（例如：“想拒绝但要委婉”、“希望表现得热情点”），让 AI 回复更贴心。
- **AI 智能回复**：配置完成后，点击“AI 回复”即可一键生成多条不同风格的回复意见供选择。
- **配置持久化**：内置大模型自定义配置区（支持自定义 **URL、模型名称、API Key**）。配置一次后自动保存，下次运行脚本无需重复填写，省时省心。

## 🚀 使用方法

1. **微信端**：在聊天窗口长按消息 -> 点击「多选」 -> 勾选需要分析的对话 -> 点击「复制」。
2. **脚本端**：打开本脚本，点击 **「自动获取剪切板」**。
3. **角色设定**：勾选自己的角色，并填写对方的身份。
4. **补充想法**（可选）：在输入框中写下你希望 AI 重点考虑的真实想法。
5. **生成回复**：点击 **「AI 回复」**，挑选最满意的回复复制使用！
